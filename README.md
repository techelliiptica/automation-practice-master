# Automation Practice Hub

A comprehensive collection of HTML practice pages designed for learning and mastering web automation testing. This repository provides hands-on scenarios covering various web automation challenges and interactions.

## üìã About This Repository

This repository contains a series of HTML practice pages that simulate real-world web automation scenarios. Each page focuses on specific web elements and interactions commonly encountered in automation testing, making it an ideal resource for learning and practicing automation frameworks like Selenium, Playwright, Cypress, and others.

## üéØ Nature of HTML Pages

The HTML pages in this repository are **educational practice pages** designed specifically for:

- **Learning Web Automation**: Practice interacting with various web elements and components
- **Testing Automation Frameworks**: Test and validate your automation scripts and frameworks
- **Skill Development**: Build proficiency in handling different types of web interactions
- **Hands-on Practice**: Gain practical experience with real-world automation scenarios

### Available Practice Scenarios

- **Form Submission** - Practice filling and submitting forms with various input types
- **Button Interactions** - Learn to interact with different types of buttons
- **Dropdown & Select** - Practice selecting options from dropdown menus
- **Checkbox & Radio** - Master handling checkboxes and radio buttons
- **Dynamic Content** - Practice waiting for and interacting with dynamically loaded content
- **Alerts & Modals** - Handle JavaScript alerts, confirm dialogs, and modal popups
- **Table Data Extraction** - Extract data from HTML tables and interact with table elements
- **File Upload** - Practice uploading single and multiple files
- **Hover & Tooltip** - Learn to handle hover effects and tooltips
- **Drag & Drop** - Practice drag and drop functionality
- **Iframe Handling** - Learn to switch between iframes and interact with nested frames
- **Date Picker** - Practice interacting with date pickers and calendar widgets
- **Link Interactions** - Practice clicking links and handling navigation
- **Image Interactions** - Verify alt text, check image sources, and extract image information
- **Super Assignment** - Comprehensive job registration form with all components

## üöÄ Getting Started

### Important: Fork Instead of Clone

**Please fork this repository instead of cloning it directly.** Forking allows you to:

- Maintain your own copy of the repository
- Make changes without affecting the original repository
- Contribute back through pull requests if desired
- Keep your practice work organized in your own GitHub account

### How to Fork

1. Click the "Fork" button at the top right of this repository page
2. Select your GitHub account as the destination
3. Once forked, clone your forked repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/automation-practice-master.git
   cd automation-practice-master
   ```
4. Open `app/index.html` in your browser to view the practice hub

### Local Setup

1. Navigate to the `app` directory
2. Open `index.html` in your web browser
3. Start practicing automation scenarios!
